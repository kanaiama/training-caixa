Warsaw Wrapper - Utilização das funções:

Construtor

Prototype

    var wrapper = new WarsawWrapper("AcronimoDoCliente", "Seed");

Parâmetros

     AcronimoDoCliente - e a identificação do cliente. Ex: GAS Tecnologia = "gas"

     Seed - informações do cliente, gerado pela ferramenta install_seed_gen


IsInstalled

Verifica se o cliente está instalado.

Prototype

    wrapper.IsInstalled(installedCallback, notInstalledCallback, errorCallback);

Parâmetros

     installedCallback - função chamada se o cliente estiver instalado.

  1. function installedCallback()

     notInstalledCallback - função chamada se o cliente não estiver instalado.

  1. function notInstalledCallback()

     errorCallback - função chamada se houver erro na chamada. Ex.: Warsaw não instalado

  1. function errorCallback(msg)

     msg - mensagem de erro





Install

Instala o cliente utilizando as informações passadas no Seed.

Prototype

    wrapper.Install(finishedInstallCallback, errorOnInstallCallback, updateTimeIdleSeconds);

Parâmetros

     finishedInstallCallback - função chamada no final da instalação.

  1. function finishedInstallCallback()

     errorOnInstallCallback - função chamada em caso de erro. Ex.: Warsaw não instalado

  1. function errorCallback(msg)

     msg - mensagem de erro

     updateTimeIdleSeconds - tempo a ser esperado, em segundos, ate que a instalação seja concluída





Update

Atualiza o cliente utilizando as informações passadas no Seed. Nenhum retorno é esperado em caso de sucesso. Em caso de erro a callback é chamada.

Prototype

    wrapper.Update(errorOnUpdateCallback);

Parâmetros

     errorOnUpdateCallback - função chamada em caso de erro. Ex.: Warsaw não instalado

  1. function errorCallback(msg)

     msg - mensagem de erro





Function10

Requisita as informações de deteção do usuário.

Prototype

    wrapper.F10(agencia, conta, usuario, parametro, desafio, comando, f10SuccessCallback, errorCallback);

Parâmetros

     agencia - agência do usuário

     conta - conta do usuário

     usuario - nome do usuário

     parametro - parâmetro definido pelo cliente

     defafio - senha utilizada para encriptar os dados retornados

     comando - comando da function10 assinado pela wscrypt

     f10SuccessCallback - função chamada em caso de sucesso.

  1. function f10SuccessCallback(response)

     response - resposta do function10

     errorOnUpdateCallback - função chamada em caso de erro. Ex.: Warsaw não instalado

  1. function errorCallback(msg)

     msg - mensagem de erro





Function13

Registra as informações usados no sentinel.

Prototype

    wrapper.F13(dataSet, comando, f13SuccessCallback, errorCallback);

Parâmetros

     dataSet - conjunto de informações a serem analisadas

  1. var dataSet = {};
  2. dataSet["name"] = "bioid";
  3. dataSet["ag"] = "2922x";

     comando - comando da function13 assinado pela wscrypt

     f13SuccessCallback - função chamada em caso de sucesso.

  1. function f13SuccessCallback(response)

     response - 1 se o cadastro foi feito com sucesso

     errorOnUpdateCallback - função chamada em caso de erro. Ex.: Warsaw não instalado

  1. function errorCallback(msg)

     msg - mensagem de erro





Encrypt

Encripta um buffer usando uma senha, o buffer e senha são passados via encryptSeed. O retorno da função é o criptograma resultante.

Prototype

    wrapper.Encrypt(command, data, user_key, salt, encryptSuccessCallback, errorCallback);

Parâmetros

     command - comando do encrypt assinado pela wscrypt.

     data - dado a ser encriptado.

     user_key - senha para a encriptação.

     salt - salgar a url. Ao habilitar isso o Encrypt tem um retorno equivalente ao Function8. Valores aceitos: 1 ou 0.

     encryptSuccessCallback - função chamada em caso de sucesso.

  1. function encryptSuccessCallback(response)

     response - resposta do encrypt()

     errorCallback - função chamada em caso de erro.

  1. function errorCallback(msg)

     msg - mensagem de erro

Info

Retorna informações sobre a maquina. Atualmente retornando os seguintes parâmetros:

     ver = product version.

     trc = quantidade de trojans.

     os_version = versão do sistema operacional.

     is_vm = se esta em maquina virtual.

Prototype

    wrapper.Info(command, infoSuccessCallback, errorCallback);

  1. function infoSuccessCallback(response)

     response - resposta do info()

     errorCallback - função chamada em caso de erro.

  1. function errorCallback(msg)

     msg - mensagem de erro





Digest

Retorna referentes ao digest. Atualmente retornando os seguintes parâmetros:

     Digest = Digest 1.

     Digest2 = Digest 2.

Prototype

    wrapper.Digest(command, digestSuccessCallback, errorCallback);

  1. function digestSuccessCallback(response)

     response - resposta do info()

     errorCallback - função chamada em caso de erro.

  1. function errorCallback(msg)

     msg - mensagem de erro
